The Impact of Latitude on Weather

This project analyzes how the weather changes relative to the distance from the equator. A dataset over 500 cities were extracted and assembled from the OpenWeatherMap API. Matplotlib was then used to plot various weather parameters vs. Latitude. The weather parameters included maximum temperature, humidity, cloudiness and wind speed.

This site provides the source data and visualizations. All observed trends and correlations were described.
